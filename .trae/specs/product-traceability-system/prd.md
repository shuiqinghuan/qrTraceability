# 农产品溯源系统 - 产品需求文档（精简版）

## 1. 产品概述

农产品溯源系统是一个轻量级的Web应用，通过直观的视觉呈现展示农产品的品种信息、种植环境和采收质量。

**设计理念：**
- 极简主义：少即是多
- 内容优先：图片和视频占据主导地位
- 快速加载：零依赖，即开即用

## 2. 核心功能

### 2.1 第一部分：产品基础信息

展示产品品种和种植信息。

**信息字段：**
- 品种名称：枣甜5号
- 品种编码：4395
- 定植地点：新疆维吾尔自治区和田地区
- 定植时间：2024-03-15

**展示方式：**
- 品种编码使用金色徽章标签突出显示
- 地点和时间使用图标+文字形式
- 卡片式布局，简洁清晰

### 2.2 第二部分：多媒体展示

以图片和视频为核心内容。

**内容类型：**
- 产品图片（2-4张）
- 产品视频（0-1个）

**展示特点：**
- 图片采用网格布局，响应式自适应
- 视频播放器带播放控制栏
- 主体位置展示，面积占页面60%以上
- 图片支持懒加载，提升性能

### 2.3 第三部分：采收质量信息

展示产品的采收和品质数据。

**数据字段：**
| 字段 | 示例值 | 展示方式 |
|------|--------|----------|
| 采收起始时间 | 2024-09-01 | 日期格式 |
| 采收终止时间 | 2024-09-15 | 日期格式 |
| 糖度 | 22.5°Brix | 数字突出显示 |
| 重量 | 15.3g | 数字突出显示 |
| 口感 | 肉质脆嫩，甜度高 | 文字描述 |
| 适应人群 | 一般人群、健康饮食者 | 标签云展示 |
| 品质小结 | 综合品质评价文字 | 引用样式 |

**展示特点：**
- 糖度和重量使用大字号突出
- 采收时间使用时间范围格式（09月01日 - 09月15日）
- 适应人群使用标签胶囊样式
- 品质小结使用引用框样式

### 3.2 视觉设计规范

**色彩方案：**
```
主色调：#4A7C59（森林绿）- 自然、健康
辅助色：#F5E6D3（暖米色）- 温馨背景
强调色：#E8A838（丰收金）- 标签、徽章
文字色：#2D3436（深灰）- 正文
背景色：#FAFAFA（浅灰白）- 页面背景
卡片色：#FFFFFF（白色）- 卡片背景
```

**圆角规范：**
```
卡片圆角：16px
图片圆角：16px
视频圆角：16px
标签圆角：9999px（胶囊形）
```

**阴影层次：**
```
卡片阴影：0 4px 16px rgba(0,0,0,0.12)
悬停阴影：0 8px 32px rgba(0,0,0,0.16)
```

**字体规范：**
```
标题字体：Noto Serif SC（思源宋体）
正文字体：Noto Sans SC（思源黑体）
标题大小：24px-32px
正文字号：14px-16px
标签字号：12px-14px
```

### 3.3 响应式设计

**布局变化：**
- **桌面端（≥1024px）**：双列网格布局
- **平板端（768px-1023px）**：双列网格
- **移动端（<768px）**：单列布局，图片横向滚动

## 6. 技术实现

### 6.1 技术选型

**框架：** Vue 3（CDN引入）
**样式：** 纯CSS（无框架）
**数据：** 内联静态数据

**无依赖，极简部署**

### 6.2 文件结构

```
/
└── index.html    # 包含所有代码的单一文件
```

**优势：**
- 零配置
- 零构建
- 一键部署
- 跨平台运行

### 6.3 部署方式

**支持的所有部署方式：**
1. GitHub Pages（免费）
2. Vercel/Netlify（免费）
3. 阿里云OSS
4. 腾讯云COS
5. Nginx/Apache服务器
6. 直接浏览器打开HTML文件

**部署步骤：**
1. 保存index.html
2. 上传到任意静态服务器
3. 访问URL即可

## 7. 性能目标
