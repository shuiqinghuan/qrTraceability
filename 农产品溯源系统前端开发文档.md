# 农产品溯源系统前端开发文档

## 文档信息

- **项目名称**：农产品溯源系统
- **前端框架**：Vue 3
- **文档版本**：V1.0
- **创建日期**：2025年
- **目标用户**：农产品溯源展示

---

## 一、项目概述

### 1.1 项目背景

农产品溯源系统旨在为消费者提供透明、可信的农产品信息展示平台。通过直观的视觉呈现，让消费者了解产品的品种信息、种植环境和采收质量，增强产品可信度和品牌价值。

### 1.2 核心功能

本系统包含三个核心功能模块：

| 模块 | 功能描述 |
|------|----------|
| **产品基础信息** | 展示品种名称、品种编码、定植地点、定植时间 |
| **多媒体展示** | 产品图片和视频的展示 |
| **采收质量信息** | 采收时间、糖度、重量、口感、适应人群、品质小结 |

### 1.3 设计理念

- **极简主义**：少即是多，保持页面简洁清晰
- **内容优先**：图片和视频占据主导地位，信息层次分明
- **快速加载**：零依赖架构，确保秒开体验
- **易于部署**：一键部署到任意静态服务器

## 二、技术架构

### 2.1 技术选型

**推荐方案：Vue 3 CDN版本**

| 技术 | 说明 | 优势 |
|------|------|------|
| Vue 3 | 通过CDN引入，无需构建 | 零配置、秒启动 |
| 纯CSS | 无CSS框架依赖 | 轻量、灵活、完全可控 |
| 静态数据 | 内联JSON数据 | 无后端、快速响应 |

### 2.2 项目结构

**极简结构（推荐）：**

```
/
└── index.html          # 包含所有代码的单一文件
    ├── HTML结构
    ├── CSS样式
    └── JavaScript逻辑
```

**标准结构（可选）：**

```
/
├── index.html          # HTML模板
├── css/
│   └── style.css       # 样式文件
├── js/
│   └── app.js          # Vue应用逻辑
└── assets/             # 静态资源
    ├── images/
    └── videos/
```

```bash
# 方式3：云存储
1. 创建OSS/COS存储桶
2. 上传index.html
3. 配置静态网站托管
4. 绑定自定义域名（可选）
```

## 三、数据结构

### 3.1 产品数据模型

```javascript
// 产品数据结构
const product = {
  // ==================== 基础信息 ====================
  id: '4395',                    // 产品唯一标识
  name: '枣甜5号',               // 品种名称
  code: '4395',                  // 品种编码
  location: '新疆维吾尔自治区和田地区',  // 定植地点
  plantingDate: '2024-03-15',   // 定植时间
  
  // ==================== 媒体资源 ====================
  images: [                      // 产品图片列表
    'https://example.com/image1.jpg',
    'https://example.com/image2.jpg',
    'https://example.com/image3.jpg',
    'https://example.com/image4.jpg'
  ],
  videos: [                      // 产品视频列表
    'https://example.com/video.mp4'
  ],
  
  // ==================== 采收质量 ====================
  quality: {
    harvestStart: '2024-09-01',  // 采收起始时间
    harvestEnd: '2024-09-15',    // 采收终止时间
    sweetness: 22.5,             // 糖度（Brix值）
    weight: 15.3,                // 重量（克）
    texture: '肉质脆嫩，甜度高，入口即化',  // 口感描述
    crowd: [                     // 适应人群
      '一般人群',
      '注重健康饮食者',
      '糖尿病患者慎用'
    ],
    summary: '得益于和田地区充足的日照和昼夜温差，今年的枣甜5号品质优于往年，糖度达到22.5°Brix，属于特级品质。'  // 品质小结
  }
};
```

### 3.2 数据字段说明

**基础信息字段：**

| 字段名 | 数据类型 | 必填 | 说明 | 示例 |
|--------|----------|------|------|------|
| id | String | 是 | 产品唯一标识 | "4395" |
| name | String | 是 | 品种名称 | "枣甜5号" |
| code | String | 是 | 品种编码 | "4395" |
| location | String | 是 | 定植地点 | "新疆和田地区" |
| plantingDate | String | 是 | 定植时间 | "2024-03-15" |

**媒体资源字段：**

| 字段名 | 数据类型 | 必填 | 说明 | 示例 |
|--------|----------|------|------|------|
| images | Array | 是 | 图片URL列表 | ["url1", "url2"] |
| videos | Array | 否 | 视频URL列表 | ["video.mp4"] |

**采收质量字段：**

| 字段名 | 数据类型 | 必填 | 说明 | 示例 |
|--------|----------|------|------|------|
| harvestStart | String | 是 | 采收开始日期 | "2024-09-01" |
| harvestEnd | String | 是 | 采收结束日期 | "2024-09-15" |
| sweetness | Number | 是 | 糖度值 | 22.5 |
| weight | Number | 是 | 重量（克） | 15.3 |
| texture | String | 是 | 口感描述 | "肉质脆嫩" |
| crowd | Array | 是 | 适应人群列表 | ["一般人群"] |
| summary | String | 是 | 品质小结 | "品质优良" |

### 10.2 使用说明

**部署步骤：**

1. **保存文件**
   - 将上述代码保存为 `index.html`

2. **本地测试**
   - 双击文件在浏览器中打开
   - 或使用本地服务器：
     ```bash
     python -m http.server 8000
     # 访问 http://localhost:8000
     ```

3. **部署上线**
   - **GitHub Pages**：
     - 创建仓库，上传文件
     - Settings → Pages → 启用
   - **Vercel/Netlify**：
     - 注册账号，连接仓库
     - 自动部署
   - **云存储**：
     - 上传到OSS/COS
     - 配置静态网站托管

**数据修改：**

修改JavaScript中的`product`对象即可更新产品信息：

```javascript
const product = ref({

});
```
